🔥 Cách cài Firebase (miễn phí)

  1. Vào console.firebase.google.com → Thêm dự án
  2. Vào Realtime Database → Tạo cơ sở dữ liệu → chọn chế độ Test
  3. Copy URL dạng: https://tên-dự-án-default-rtdb.firebaseio.com
  4. Mở file script.js, tìm dòng đầu và dán vào FIREBASE_URL
const FIREBASE_URL = 'https://your-app-default-rtdb.firebaseio.com';

Sau đó lưu và reload — dữ liệu sẽ tự đồng bộ trên mọi trình duyệt! 🎉
