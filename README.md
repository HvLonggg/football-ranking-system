# ⚽ Football Group Standings Calculator

A simple web application that calculates football group standings based on match results.

Users can input match scores and the system will automatically calculate:

- Matches played
- Wins, draws, losses
- Goals for / goals against
- Goal difference
- Points
- Team rankings

The standings are sorted automatically based on standard football tournament rules:
1. Points
2. Goal difference
3. Goals scored

## 🛠 Technologies

- HTML
- CSS
- JavaScript

## ▶ Run Project

Simply open:

index.html

in your browser to start using the application.
